Para executar os códigos do documento JS você precisa usar o atalho [ CTRL + ; ] ("descomentar") na linha da tag script correspondente a cada arquivo que fica dentro da tag '< head >' .


Exemplo: se escolher o arquivo "condicionais.js", então descomente a linha correspondente a ele no html, no caso a linha 10.


Não esqueça, que para poder ver a execução do código pelo navegador web você precisa abrir o DevTools.

Caso não saiba como, use o atalho [ CTRL + SHIFT + J ] ou simplesmente [ F12 ].
